# Vietnamese Title & Variant Style — TikTok Shop Vietnam

## Title rules (column B)

1. Translate the Chinese title into natural Vietnamese a local shopper reads
   fluently. Not word-for-word — local buying-search phrasing.
2. **≤80 characters** hard cap. If over, trim the least important descriptors
   while keeping: product noun + key differentiators. Never exceed 80.
3. **Remove** these tokens entirely: `原装` (genuine/original), `原厂` (factory
   original), and any brand name (Nike, Apple, Sony, etc.).
4. If the source title brands a product (e.g. "苹果手机壳"), rephrase as
   **compatible-with** form: `ốp lưng phù hợp với iPhone` ("case compatible with
   iPhone"). Pattern: `phù hợp với [thương hiệu] [mẫu mã]`. This signals
   compatibility without claiming to be the brand.
5. Lead with the product category noun (Vietnamese shoppers search noun-first):
   `Ốp lưng điện thoại...`, `Đệm tựa tay ô tô...`.
6. Use Vietnamese diacritics correctly.

### Example

Source: `汽车扶手箱垫中央增高垫车用中间保护套车载肘托高级加厚加长通用`
→ `Đệm tựa tay cạnh cần số ô tô, tấm nâng cao bảo vệ tựa tay, dày dài通用`
→ refined: `Đệm tựa tay ô tô tấm nâng cao bảo vệ tựa tay dày dài đa năng` (≤80)

## Variant attribute names (G/I/K) — standard translations

| Chinese | Vietnamese |
|---------|-----------|
| 颜色分类 | Phân loại màu |
| 商品规格 | Quy cách sản phẩm |
| 尺码 | Kích cỡ |
| 容量 | Dung tích |
| 材质 | Chất liệu |
| 款式 | Kiểu dáng |
| 套餐 | Gói sản phẩm |
| 型号 | Mẫu mã |

If a name isn't listed, translate literally and naturally.

## Variant attribute values (H/J/L)

- Remove any brand name embedded in the value.
- Translate the actual content (color, spec, size, style).
- Marketing fluff like `**甄选皮革**耐用十年**` → keep concise, translate the
  substance: `da cao cấp, bền`. Drop pure hype.

## IP-safe / infringement avoidance

- Never reproduce a brand's exact trademarked name as if it IS the product.
  Use `phù hợp với` (compatible with) for brand-fit items.
- Avoid counterfeit-suggesting words: drop `正品`, `高仿`, `1:1`, `原单`.
- Don't include competitor store names or handles.
- Generic descriptors (style, material, function) are fine.

## TikTok Vietnam listing compliance notes

- Title ≤80 chars (platform limit for many categories).
- No contact info / social handles in title or description text.
- Description is image-based here (`<img>` HTML) — keep only product + spec
  images, drop promo/after-sales (see image-rules.md).
- Currency VND, stock set to 30 per the user's standing instruction.
